## 재검증 생략 가능 (바로 Step 5)
- 설명 문구 수정
- 에이전트 이름 변경
- 브리핑 표현 수정

## Step 4 재실행 필수
- 에이전트 추가 또는 삭제
- 에이전트 역할·책임 변경
- tools 권한 변경
- 공통 skills·hooks 변경
- 보안 책임 변경
- Bash·Write·Edit·외부 MCP 권한 추가

재검증 필수 항목이 하나라도 포함되면:
설계서를 v{N+1}로 저장 → Step 4 재실행
(design_loop는 증가하지 않음, 사용자 직접 승인 수정이므로)
